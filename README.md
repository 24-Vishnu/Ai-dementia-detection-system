**AI TOOL FOR EARLY-STAGE DEMENTIA DETECTION**

An AI-powered healthcare web application developed for early dementia screening using speech analysis, cognitive assessment, and machine learning techniques.

**Features**
- Speech-based analysis
- Cognitive assessment support
- Machine learning prediction
- Interactive web interface
- Data preprocessing & visualization

**Technologies Used**
- Python
- Flask
- HTML, CSS, JavaScript
- Pandas, NumPy, Scikit-learn
- SQLite
